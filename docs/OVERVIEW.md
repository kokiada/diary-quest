# DiaryQuest (ダイクエ) - プロジェクト概要

## プロジェクト紹介

DiaryQuest（ダイクエ）は**Flutterベースのゲーム化日記アプリ**で、日々の日記エントリーをRPGスタイルのクエストに変換します。AIパワーによる分析とゲーム要素を組み合わせることで、個人の成長追跡を魅力的でモチベーションの高いものにします。

### コアコンセプト

ユーザーが日々の経験を話す → AIが経験を分析 → 経験が15の成長パラメーターの経験値に変換される → ユーザーはRPGスタイルのレベルアップシステム（ジョブ、スキル、相棒ナビゲーター付き）で進行する

### ビジョン

*"日々の生活を冒険に変える。すべての経験がクエストになり、すべての実績が経験値になり、毎日が成長の機会になる。"*

## 差別化要因

### 1. 音声ファースト体験
- **音声認識統合**：自然な日記入力
- タイピング不要 - 話すだけ
- 日本語UIを含む多言語サポート

### 2. AIパワーによる分析
- **GPT-4o統合**：インテリジェントな分析
- ありふれた報告を構造化された成長機会に変換
- パーソナライズされたリフレーミングとインサイトを提供

### 3. RPGゲーム化
- **MECEフレームワークに基づく15の成長パラメーター**
- **5つのナビゲーター性格**（軍師/アニキ/精霊/騎士/猫妖精）が独自のフィードバックを提供
- **ジョブとスキルシステム**で進行状況を追跡
- **レベルアップメカニクス**と視覚的な進行表示

### 4. MECE成長フレームワーク
15のパラメーターは5つのカテゴリーに組織されています：
- **思考（Tactics）**：分析力、創造力、専門性
- **実行（Execution）**：スピード、品質、仕組み化
- **対人（Social）**：交渉力、共感力、言語化力
- **自制（Mastery）**：レジリエンス、規律、メタ認知
- **挑戦（Frontier）**：リスクテイク、学習棄却、ビジョン

## 結論

DiaryQuestは、ゲーム化とAIを通じて個人の開発に新しいアプローチを提供します。日々の振り返りを構造化された成長機会に変えることで、個人の開発を魅力的で追跡可能にします。音声入力、AI分析、RPG要素の組み合わせは、従来の日記アプリとは一線を画すユーザー体験を作り出します。

このプロジェクトの技術実装は、クリーンアーキテクチャ、適切な状態管理、スケーラブルなバックエンド統合など、最新のFlutter開発プラクティスを活用しており、将来の成長と機能拡張に備えた基盤を構築しています。
