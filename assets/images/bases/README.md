# Base Assets

このディレクトリにはプレイヤーのベース（アバター）画像を配置します。

## 必要な画像

- `base_level1.png` - 初期ベースレベル1
- `base_level2.png` - ベースレベル2
- `base_level3.png` - ベースレベル3

## 画像仕様

- サイズ: 512x512px (推奨)
- フォーマット: PNG (透明背景付き)
- アスペクト比: 1:1 (正方形)
- スタイル: ピクセルアート調 (16x16 or 32x32 ベース)

## 配置手順

1. 画像ファイルを準備する
2. このディレクトリに配置する
3. アプリを再ビルドする (`flutter run`)

## 注意事項

- 画像ファイルはGitにコミットされます
- 画像サイズは適切に最適化してください
