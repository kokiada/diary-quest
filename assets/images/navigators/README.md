# Navigator Images

このディレクトリには5種類のナビゲーター画像を配置します。

## 必要な画像

- `strategist.png` (軍師)
- `aniki.png` (アニキ)
- `spirit.png` (スピリット)
- `knight.png` (騎士)
- `cat.png` (猫)

## 画像仕様

- サイズ: 512x512px (推奨)
- フォーマット: PNG (透明背景付き)
- アスペクト比: 1:1 (正方形)

## 配置手順

1. 画像ファイルを準備する
2. このディレクトリに配置する
3. アプリを再ビルドする (`flutter run`)

## 注意事項

- 画像ファイルはGitにコミットされます
- 画像サイズは適切に最適化してください
