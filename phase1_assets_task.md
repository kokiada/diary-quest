# Asset Creation Task

## Objective
Create placeholder image assets for DiaryQuest app.

## Required Actions

1. **Navigator Assets**
   - Create placeholder images for 5 navigator personalities:
     - 軍師 (strategist)
     - aniki (big brother)
     - spirit
     - knight
     - cat

2. **Base Assets**
   - Create placeholder images for player bases/avatars
   - Create evolution stage placeholders (if needed)

3. **UI Assets**
   - Create placeholder icons for:
     - Settings
     - Quest button
     - Level up indicators
     - Skill unlock badges
     - Job unlock badges

4. **Lottie Animations**
   - Create placeholder Lottie animations for:
     - Level up animation
     - Skill unlock animation
     - Job unlock animation
     - Base evolution animation

## Requirements
- Use consistent pixel art style
- Create appropriate sizes for iOS (2x, 3x)
- Organize assets in proper directory structure
- Update pubspec.yaml with asset references

## Deliverables
- Complete asset package in `assets/` directory
- Updated pubspec.yaml with asset declarations
- Asset usage documentation
- Recommendations for final asset production
